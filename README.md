# AI-Assisted Workflow — Linear & Claude at Arkyn

A practical walkthrough for the Arkyn team on how we use Linear for project tracking and Claude as an AI co-pilot across product, development, and QA.

## View the walkthrough

Open `index.html` in your browser, or visit the GitHub Pages deployment.

## What's covered

- **Product & Planning** — capturing requirements, writing specs, breaking into tasks
- **Development** — implementing with Claude Code, creating PRs, code review
- **Cross-Platform Handoff** — using iOS implementations to define Android work (and vice versa)
- **Testing & QA** — manual testing with Claude in Chrome, automated UI testing with agent-device
- **Integrations** — Sentry crash monitoring, Zendesk customer tickets, GitHub auto-linking
- **Claude Touchpoints** — Claude Code (devs), Cowork (PMs), Claude in Chrome (testers & support)
